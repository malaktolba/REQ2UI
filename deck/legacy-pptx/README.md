## How to edit and build
The `.pptx` is a **native, fully editable PowerPoint file** (real text boxes and
shapes — not flat images), generated by the GLM slide agent. To update:

1. Open `Req2UI-Defense.pptx` in PowerPoint (or Google Slides / LibreOffice Impress).
2. Edit text, swap images, reorder slides as normal.
3. Re-export the PDF: **File → Export → Create PDF/XPS** → save to `exports/`.

Small text/factual fixes can also be patched directly in the underlying XML
(`ppt/slides/slideN.xml` inside the zip) — that is how the fixes below were applied —
but **layout changes are far easier in PowerPoint.**

> If you regenerate the deck from the GLM agent again, it will overwrite these edits.
> Keep this file as the source of truth once you start editing here.
